# Student Performance Prediction System

This project is a machine learning system designed to predict student academic performance based on various factors such as study time, attendance, and past scores.

## Features

* Data preprocessing
* Model training
* Performance prediction
* Evaluation metrics

## Setup

1. Clone the repository

```
git clone https://github.com/NathanBvumbwe/student_perfomance_system.git
```

2. Create environment

```
conda create -n student_perf_env python=3.11
conda activate student_perf_env
```

3. Install dependencies

```
pip install -r requirements.txt
```

## Run the system

```
python main.py
```
